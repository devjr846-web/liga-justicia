# Historia de flash


Flash es un término que se refiere principalmente al famoso superhéroe de DC Comics con supervelocidad, a un destello o dispositivo de iluminación fotográfica, o a la antigua plataforma de software multimedia de Adobe.1. El superhéroe de DC ComicsFlash (también conocido como El Destello) es el nombre de varios personajes de historietas. El más conocido es Barry Allen, el "Corredor Escarlata". El personaje tiene el poder de súper velocidad, lo que le permite moverse, correr y pensar a una velocidad
